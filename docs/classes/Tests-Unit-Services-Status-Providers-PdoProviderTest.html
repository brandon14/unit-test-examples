Class PdoProviderTest.

PdoProvider tests.

It's easy to imagine testing a class that uses a database connection (in this case an connection) would actually connect to a database. But by passing in the connection object to the class, we can mock that object in our tests below so we never have to leave the application boundary to test this class. As long as we force our mock to behave as a normal class would (think adhering to the documentation for the class), then we can be sure our class behaves in accordance given our assumption of how the class behaves.
